#SimpleNotepad
SimpleNotepad is text editor that allow you to edit un-styled text.
The editor supports the folowing features:
- Text input;
- Cursor selection (mouse and arrow keys);
- Newlines;
- Backspace;
- Undo and redo;
- Open/Save text file;
- Window re-sizing;
- Scrollbar (vertical);
- Changing font size;

SimpleNotepad uses javafx library.
